## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report vulnerabilities through
[GitHub Security Advisories](../../security/advisories/new)
or by emailing `[INSERT CONTACT EMAIL]`.

Include a description of the vulnerability, steps to reproduce, and
the affected {{PROJECT_NAME}} version(s). We will respond as soon as
we can and coordinate disclosure with you before any public announcement.

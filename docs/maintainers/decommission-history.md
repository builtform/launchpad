# Decommission History

> **Purpose.** Append-only audit log of file paths permanently removed from
> the LaunchPad repository. Every deletion of a CODEOWNERS-protected path,
> a user-facing surface, or a documented architectural element MUST add a
> same-commit entry here. The audit-log lint rule (Phase 8.5 Slice E)
> enforces same-commit pairing for future deletions; Phase 8 entries below
> are the seed corpus.
>
> **Schema rules.** Append rows below the table; never edit historical
> rows. If a deletion is reverted, append a new "RESTORED" row referencing
> the original entry rather than deleting that entry. The Reviewer field
> records the human GitHub username of the maintainer who approved the
> deletion. Replacement is a marketplace-install command, slash-command
> name, kernel template name, or "none; v2.2 BL" if the behavior is
> intentionally not preserved at v2.1.

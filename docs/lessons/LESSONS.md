# Lessons Learned

Categorized learnings from development iterations. Each entry in `docs/solutions/` represents a reusable pattern or insight discovered during compound loops.

## Index

See `docs/solutions/` for categorized learnings with YAML frontmatter.

New learnings are added automatically by compound-engineering agents during `/workflows:compound` and `/workflows:review` cycles. Manual entries are welcome.

## Format

Each file in `docs/solutions/` follows this format:

```yaml
---
title: "Descriptive title"
category: "frontend|backend|infra|testing|workflow"
date: YYYY-MM-DD
---
```

Followed by markdown content describing the pattern, gotcha, or insight.

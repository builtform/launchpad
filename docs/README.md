# Documentation

This directory contains all project documentation.

| Directory         | Purpose                                                   |
| ----------------- | --------------------------------------------------------- |
| `architecture/`   | System design documents (PRD, tech stack, app flow, etc.) |
| `plans/`          | Implementation plans and task breakdowns                  |
| `solutions/`      | Categorized learnings from compound loops                 |
| `reports/`        | Investigation reports, audits, postmortems                |
| `brainstorms/`    | Brainstorming and exploration docs                        |
| `experiments/`    | Spike summaries and prototype notes                       |
| `lessons/`        | Append-only log of learnings                              |
| `archive/`        | Retired docs (read-only, never delete)                    |
| `tasks/`          | Active work tracking                                      |
| `ui/ux/`          | Design system documentation                               |
| `handoffs/`       | Session handoff notes and implementation summaries        |
| `guides/`         | How-to guides and onboarding walkthroughs                 |
| `skills-catalog/` | Curated skill recommendations for downstream projects     |
| `consultants/`    | Consultant-style advisory docs                            |
| `eval/`           | Skill and agent evaluation results                        |
| `articles/`       | Reference articles and external content                   |
